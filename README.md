# SIRONIC IntraSystem

Internal CRM and Partner Portal monorepo scaffold with strict TypeScript, modular package boundaries, and RBAC-first architecture.

## Apps

- `apps/crm`
- `apps/partner-portal`

## Packages

- `packages/shared`
- `packages/ui`
- `packages/lib`
- `packages/types`
- `packages/auth`
- `packages/rbac`
- `packages/modules`

## Quick Start

1. `pnpm install`
2. `pnpm dev`

## Quality Gates

- `pnpm typecheck`
- `pnpm test`
- `pnpm build`
